1. Obtain `Unihan_IRGSources.txt` from <<https://www.unicode.org/Public/UCD/latest/ucd/Unihan.zip>>

2. Run `./extract.py`.
3. Run `grep -P 'code_point|,' unihan-radical-strokes-readable.tsv > unihan-radical-strokes-readable-ambiguous.tsv`.
